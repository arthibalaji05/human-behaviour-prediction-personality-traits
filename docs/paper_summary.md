# Research Publication

## Predicting Human Behaviour with Human Traits

### Authors
Arthi B, Arul Venil, Dr. M.D. Kamalesh

### Published
Grenze International Journal of Engineering and Technology (GIJET), 2025

## Overview

This research explores human behaviour prediction using machine learning techniques based on personality traits.

The project uses:
- Principal Component Analysis (PCA)
- K-Means Clustering
- Data Visualisation
- Tkinter GUI

## Methodology

### Data Processing
- Data cleaning
- Feature preparation
- Personality trait analysis

### Machine Learning
- PCA was used for dimensionality reduction.
- K-Means clustering was applied to identify behavioural patterns.

### Application
A Tkinter-based GUI was developed for interactive personality assessment.
